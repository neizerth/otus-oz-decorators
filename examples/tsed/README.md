# Примеры декораторов в TS.ed

Здесь собраны примеры использования декораторов в фреймворке TS.ed.

## Что такое TS.ed?

TS.ed - это TypeScript фреймворк для создания backend-приложений, который полностью построен на декораторах. Он предоставляет:
- Dependency Injection через декораторы
- Роутинг через декораторы
- Middleware через декораторы
- И многое другое

## Запуск

```bash
npm run dev:tsed
```

Или напрямую:
```bash
ts-node-dev --respawn --transpile-only examples/tsed/index.ts
```
