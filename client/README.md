# Restaurant Client

React клиент для системы заказов ресторана.

## Запуск

```bash
npm install
npm run dev
```

Приложение будет доступно по адресу `http://localhost:5173`

## Использование

Откройте в браузере адрес `http://localhost:5173/{tableId}`, где `tableId` - номер столика (1-4).

На странице можно:
- Просмотреть список доступных блюд
- Выбрать блюда для заказа (чекбоксы)
- Оформить заказ выбранных блюд

## API

Клиент использует API бэкенда на `http://localhost:3000/api`
